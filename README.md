# Dear Passenger — Turbulence

A browser-based Three.js game built with Vite.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Import the GitHub repository in [Vercel](https://vercel.com/new).
2. Keep the automatically detected Vite configuration.
3. Deploy. Vercel will run `npm run build` and publish `dist`.

No server, environment variables, or extra deployment configuration is required.
